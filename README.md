# Hola! 西班牙语单词卡片 (¡Hola! Tarjetas de Español)

这是一个专为西班牙语学习者设计的**轻量级、无广告、极简、高保真**的渐进式单词记忆与卡片复习应用。该应用采用**第一性原理**构建，专注于最高效、最纯粹的语言内化，拒绝一切花哨的无用功。

## 🧭 用户关切与关键解答

### 1. 关于 Local Storage (浏览器本地存储) 的路径与数据保障
为了最大化保障隐私、实现秒开，本应用在不连接外部云端数据库时，默认将所有词库和复习记录存储于您浏览器的 **LocalStorage** 中。
* **物理存储路径**：
  * **Windows (Chrome)**: `%LocalAppData%\Google\Chrome\User Data\Default\Local Storage\leveldb`
  * **macOS (Chrome)**: `~/Library/Application Support/Google/Chrome/Default/Local Storage/leveldb`
  * **Safari (macOS)**: `~/Library/Safari/LocalStorage`
* **防丢失建议（重点）**：
  * **不要轻易清空浏览器缓存或执行垃圾清理**：一些系统清理软件（如 CCleaner）或浏览器的“清除所有历史记录”可能会顺带清除 LocalStorage。
  * **定期使用“导出功能”**：我们在总词库页面右上角为您定制了**一键导出**功能（支持 CSV 和 Markdown），建议您每周或在大量录入后进行手动备份，随时导出的文件即是您的永久数据资产。

### 2. 关于网络与代理要求 (Vercel 部署后)
本应用是一套**完全去中心化、端到端本地运行**的高性能 React 应用：
* **无 API 调用**：本应用不调用任何第三方的 Gemini API 或是额外的后端微服务。
* **无需翻墙/代理**：在 Vercel 部署成功后，只要能打开部署地址，后续的所有单词卡片展示、记忆状态计算、本地库筛选均在您本地浏览器中进行，**完全不需要连接外部API，不需要开启任何代理**。
* **语音发音说明**：语音朗读（TTS）功能使用的是您浏览器原生搭载的 **Web Speech API**。现代操作系统（iOS、macOS、Android、Windows）均自带高品质的西班牙语本地离线发音引擎（如 macOS 的 Jorge 或 Paulina）。因此，**即使在断网状态下，只要您的设备系统支持，依然可以正常发音**，且完全免费。

### 3. 关于 24 点过了未刷新的原理解析与修复
我们以**第一性原理**溯源并修复了该问题：
* **原逻辑缺陷**：先前的判断机制依靠 `new Date().toISOString().split("T")[0]`。由于 `toISOString()` 默认采用 **UTC 零时区**，导致在东八区（北京时间 UTC+8）深夜 24:00 (即 00:00) 到早晨 08:00 期间，浏览器获取的日期依然停留在前一天（例如 6月29日23点 对应的 UTC 还是 6月29日；而 6月30日01点 对应的 UTC 会被解析为 6月29日17点）。这就导致了打卡日历无法点亮 6月30日，且“今日复习任务”误判当前依然是 6月29日 而拒绝生成新任务。
* **根本性修复**：
  1. 我们构建并统一了全工程的 **本地时区格式化函数 (`getTodayDateString()`)**，该函数严格按照用户设备的系统本地时间（年-月-日）进行字符串提取，确保过了 24 点立刻在本地触发物理天数递增。
  2. 修复了打卡日历（StatisticsView）中计算起始日期和判断“今天”的逻辑，确保本地时间打卡瞬间与热力图格子 100% 对齐。
  3. 完善了跨天检测监听器（Midnight Checker），在后台运行时即使设备不刷新，每过 10 秒也会检测一次日期，过了 24 点会自动清空当日已复习 ID 缓存，自动重算并呈现全新的一组待复习单词！

  * 拒绝填鸭式词书！我们坚持“**只有自己亲手录入的单词才能真正内化**”这一第一性语言学习原理。

## 🛠 功能架构

### 1. 记忆卡片区 (Flashcards Panel)
* 呈现今日亟需复习的卡片。支持点击或按空格键翻面。
* 提供“认识 (Know - 熟悉度+1)”与“不认识/模糊 (Hard/No - 熟悉度重置/降低)”按钮。
* 支持一键播放西班牙语发音。
* 内置**自动轮播切换**，解放双手。

### 2. 总词库管理 (Word Bank)
* **高级双向排序控制**：
  * 🌟 **首创双向极简箭头键**：同时显示上升 `↑` 和下降 `↓` 箭头。当进行**降序**排列时，`↓` 处于激活的高亮沙绿色状态，而 `↑` 变灰；当切换为**升序**时，`↑` 激活高亮，`↓` 变灰。直观清晰。
  * **时间排序**：按录入的先后顺序进行追溯。
  * **熟悉度排序**：优先复习最不熟悉的词，或者筛选出已经滚瓜烂熟的单词。
* **多维度检索**：支持对单词、中文翻译、英文翻译及例句进行全文本实时模糊搜索。
* **数据导出**：右上角下载按钮一键生成 Markdown 文档或标准 CSV 数据。
* **编辑与删除**：对拼写错误的单词或翻译，支持随时就地修改与移除。

### 3. 本地手写录入 (Manual Entrance)
* 全新重构的宽幅录入卡片。
* 输入框布局经过严格的栅格对齐设计，完美兼容手机和宽屏设备。

### 4. 科学复习规则配置 (Spaced Rules)
* **复习上限**：可自定义每日最多复习的卡片数量。
* **重现频率**：支持按传统艾宾浩斯间隔复习，或使用线性复习。
* **优先级队列**：可选择优先温习“熟悉度极低 (救火模式)”还是“接近遗忘临界点”的词卡。

## ⚠️ 使用须知与最佳实践

1. **重音符号的录入**：西班牙语包含特殊的重音字母（如 `á, é, í, ó, ú, ü, ñ, ¡, ¿`）。录入时请使用西班牙语键盘布局，或直接复制粘贴。在导出 CSV 文件时，我们已对这些字符做了完美兼容，您可放心使用 Microsoft Excel 或 macOS Numbers 导入打开。
2. **浏览器隐私模式警告**：切勿在**无痕模式 (Incognito/Private Browsing)** 下长期使用该应用，因为无痕模式会在浏览器关闭时强制清空所有的 LocalStorage 记录。
3. **不要使用具有强力清空性质的插件**：某些一键清理垃圾的浏览器扩展会清理 LocalStorage，请将本应用的网址加入到清理白名单中。
4. **例句高亮小技巧**：在录入例句时，系统会自动寻找并对与上方“单词”框中拼写一致的词进行**加粗展示**。例如：录入单词为 `hablar`，例句中写有 `hablar`，在复习卡片背面的例句中就会自动渲染为 **hablar**。
